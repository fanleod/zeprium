Layout & Spacing

Our layout system uses consistent spacing and container widths:

  • Container Width: 100% (max-width: 1200px)
  • Content Box Width: 600px
  • Grid Gaps: 2rem
  • Section Margins: 3rem

Responsive Design

Our design system adapts to different screen sizes:

  • Desktop: > 768px (16px base font size)
  • Tablet: 480px - 768px (15px base font size)
  • Mobile: < 480px (14px base font size)

Key breakpoints:

  • Grid Layout: Adapts to screen size
  • Navigation: Scrollable on mobile
  • Typography: Scales proportionally
  • Spacing: Reduces on smaller screens
